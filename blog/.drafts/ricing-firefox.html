I) Homepage/New Tab Page

Relevant Links: /wg/ thread, a pastebin, Startpage Emporium
To start off, we will be reviewing one of the two of these four sections which is not specific to Firefox: homepages. The concept is very simple: a webpage which is functionally useful when you open a blank Firefox window. It may include anything. Common elements include:
-Links to frequently visited sites
-Search bar
-Weather
-Autistic /biz/ crypto prices
-Waifus telling you to have a nice day
Point is, there is plenty of creativity you can use here, and its not inoften that I will see someone do something new and cool in a homepage. To create a basic, /wg/ ready homepage, you can simply make one using HTML+CSS (although you may need to write JS scripts for some things, like weather). This can be done fairly easily. If you're lazy, you can perform the age-old art of stealing other people's shit. See above, there are tons and tons of homepages people make. I'm currently using one I stole off of someone else (and changed a bit) but I am appaled at how bloated it is and I will probably write my own soon. Once you have a homepage you like, it's time to apply it. You can host the html file wherever you want. I reccomend hosting it on Github Pages . Once it's hosted, go to about:preferences, and change Homepage and new windows to Custom URLs.... This is all you need to do if you just want to change your homepage. However, chances are, you also want to change your newtab. Unfortunately, Firefox removed our ability to do custom newtabs a few updates ago, so I use the plugin Custom New Tab to change the newtab page. Just set the newtab to your hosted file, and you should be good to go.

II) Stylus

Relevant Links: Userstyles
In case you're unaware, Stylus is a plugin for Firefox and Chrome that lets you use your own custom CSS on websites. The tool is pretty cool and can lead to some very nice site designs. Stylus also lets you use Global Styles, which is css that will apply to every website you visit. While it may sound good to have a consistent style among all websites you visit, you soon realize that Global Styles are autistic and will make 80% of websites look like garbage (seriously, just don't use them). Admittedly, Stylus can be somewhat difficult to use sometimes. It requires you to study the website's CSS and mostly involves a lot of tinkering. I recommend writing CSS for only websites you frequent, if you value your time. Of course, userstyles.org has a ton of good styles you can steal.
